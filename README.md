# Sequence Analysis of VEGFR-2 using BLAST, MSA and Phylogenetic Tree

## About the Project
This project focuses on comparative analysis of VEGFR-2 protein sequences across different species.

## What I Did

- Collected VEGFR-2 protein sequences of human, mouse and rat from NCBI  

- Performed BLASTp using the human sequence as reference to analyze sequence similarity  
- Recorded and analyzed similarity results  

- Performed Multiple Sequence Alignment (MSA) using Clustal Omega  
- Identified conserved regions among human, mouse and rat sequences  

- Generated a phylogenetic tree to study evolutionary relationships between species  

## Tools Used
- NCBI BLAST  
- Clustal Omega  

## Conclusion
The analysis helped in understanding sequence similarity, conserved regions and evolutionary relationships of VEGFR-2 across species.
